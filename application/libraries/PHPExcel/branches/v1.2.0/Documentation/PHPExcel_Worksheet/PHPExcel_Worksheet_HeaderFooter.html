PHPExcel_Worksheet_HeaderFooter
 
  1.  Header/Footer Formatting Syntax taken from Office Open XML Part 4 - Markup Language Reference, page 1970:
    2. 

  2.  
    3. 

  3.  There are a number of formatting codes that can be written inline with the actual header / footer text, which
    4. 

  4.  affect the formatting in the header or footer.
    5. 

  5.  
    6. 

  6.  Example: This example shows the text "Center Bold Header" on the first line (center section), and the date on
    7. 

  7.  the second line (center section).
    8. 

  8.          &CCenter &"-,Bold"Bold&"-,Regular"Header_x000A_&D
    9. 

  9.  
    10. 

  10.  General Rules:
    11. 

  11.  There is no required order in which these codes must appear.
    12. 

  12.  
    13. 

  13.  The first occurrence of the following codes turns the formatting ON, the second occurrence turns it OFF again:
    14. 

  14.  - strikethrough
    15. 

  15.  - superscript
    16. 

  16.  - subscript
    17. 

  17.  Superscript and subscript cannot both be ON at same time. Whichever comes first wins and the other is ignored,
    18. 

  18.  while the first is ON.
    19. 

  19.  &L - code for "left section" (there are three header / footer locations, "left", "center", and "right"). When
    20. 

  20.  two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the
    21. 

  21.  order of appearance, and placed into the left section.
    22. 

  22.  &P - code for "current page #"
    23. 

  23.  &N - code for "total pages"
    24. 

  24.  &font size - code for "text font size", where font size is a font size in points.
    25. 

  25.  &K - code for "text font color"
    26. 

  26.  RGB Color is specified as RRGGBB
    27. 

  27.  Theme Color is specifed as TTSNN where TT is the theme color Id, S is either "+" or "-" of the tint/shade
    28. 

  28.  value, NN is the tint/shade value.
    29. 

  29.  &S - code for "text strikethrough" on / off
    30. 

  30.  &X - code for "text super script" on / off
    31. 

  31.  &Y - code for "text subscript" on / off
    32. 

  32.  &C - code for "center section". When two or more occurrences of this section marker exist, the contents
    33. 

  33.  from all markers are concatenated, in the order of appearance, and placed into the center section.
    34. 

  34.  
    35. 

  35.  &D - code for "date"
    36. 

  36.  &T - code for "time"
    37. 

  37.  &G - code for "picture as background"
    38. 

  38.  &U - code for "text single underline"
    39. 

  39.  &E - code for "double underline"
    40. 

  40.  &R - code for "right section". When two or more occurrences of this section marker exist, the contents
    41. 

  41.  from all markers are concatenated, in the order of appearance, and placed into the right section.
    42. 

  42.  &Z - code for "this workbook's file path"
    43. 

  43.  &F - code for "this workbook's file name"
    44. 

  44.  &A - code for "sheet tab name"
    45. 

  45.  &+ - code for add to page #.
    46. 

  46.  &- - code for subtract from page #.
    47. 

  47.  &"font name,font type" - code for "text font name" and "text font type", where font name and font type
    48. 

  48.  are strings specifying the name and type of the font, separated by a comma. When a hyphen appears in font
    49. 

  49.  name, it means "none specified". Both of font name and font type can be localized values.
    50. 

  50.  &"-,Bold" - code for "bold font style"
    51. 

  51.  &B - also means "bold font style".
    52. 

  52.  &"-,Regular" - code for "regular font style"
    53. 

  53.  &"-,Italic" - code for "italic font style"
    54. 

  54.  &I - also means "italic font style"
    55. 

  55.  &"-,Bold Italic" code for "bold italic font style"
    56. 

  56.  &O - code for "outline style"
    57. 

  57.  &H - code for "shadow style"
    58.
